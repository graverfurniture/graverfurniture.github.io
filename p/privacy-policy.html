3. Android Permission Used

The app requires the following Android permission for proper operation:

<uses-permission android:name="android.permission.READ_EXTERNAL_STORAGE"
    android:maxSdkVersion="32" />

Permission Explanation:

  • READ_EXTERNAL_STORAGE — This permission allows the app to read files stored on your device (for example, to import data or display user-selected files). It is limited to Android 13 (API level 32) and below, following Google Play’s latest privacy requirements.

The permission is used solely for app functionality and never for collecting, sharing, or selling user data. GF Task does not access any private or personal files without user interaction.
